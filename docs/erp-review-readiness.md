# ERP Review Readiness

## Current Scope

- Task review comments now support follow-up notes, edits, soft deletes, and audit history.
- Comment history is kept on the task so the approval flow can show the latest comment by default while preserving older versions for permitted users.
- The current app remains focused on approval flow, task upload, review routing, assigned contributors, users, and campaign scheduling.

## Future Scope

- Product catalog and bundle definitions.
- Order metadata, reporting fields, and expandable reporting views.
- Warehouse zones with configurable zone names and rules.
- CRM, procurement, marketing, website, loyalty, and promo-code modules.

## Next Review Checks

- Confirm comments can be continued after a review action is submitted.
- Confirm only permitted users can edit, delete, and view edit history.
- Confirm deleted comments hide content from normal view while preserving audit data.
- Keep future ERP modules separate from the current approval-flow logic until their workflows are defined.
